getPath(path) → {any|null}

Source:
  • quercus.esm.js, line 159
Since:
  • 1.0.0
Returns value of a given path. If the path could not be found, null is returned
Example
const q = new Quercus([
      [["foo", "bar"], 5],
      [["foo", "bizz"], 12],
      [["bar", "fazz"], 560]
  ]);

// => 5
q.getPath(["foo", "bar"]);

// =>  Quercus{"fazz"=> 560}
q.getPath(["bar"]);

// => null
q.getPath(["lorem"]);

setPath(path, val) → {Quercus|null}

Source:
  • quercus.esm.js, line 188
Since:
  • 1.0.0
Sets value of a given path. If the given path is empty, null is returned. If the value was set successfully, the value's Node is returned.
Example
const q = new Quercus();

 // => Quercus{"bar"=>5}
q.setPath(["foo", "bar"], 5);

// => Quercus{"fazz"=>560}
q.setPath(["bar", "fazz"], 560);

// => null
q.setPath([], "foo");
