Checks if a value is a Quercus instance.

Example
const q = new Quercus([["foo", "bar"], 5]);

Quercus.isQuercus(q)
// => true

Quercus.isQuercus(q.getPath(["foo"]))
// => true

Quercus.isQuercus("foo")
// => false

getPath(path) → {any|null}

Source:
  • quercus.esm.js, line 162
Since:
  • 1.0.0

Returns value of a given path.

If the path could not be found, null is returned.

Example
const q = new Quercus([
      [["foo", "bar"], 5],
      [["foo", "bizz"], 12],
      [["bar", "fazz"], 560]
  ]);

q.getPath(["foo", "bar"]);
// => 5

q.getPath(["bar"]);
// => Quercus{"fazz": 560}

q.getPath(["lorem"]);
// => null

setPath(path, val) → {Quercus|null}

Source:
  • quercus.esm.js, line 192
Since:
  • 1.0.0

Sets value of a given path.

If the given path is empty, null is returned. If the value was set successfully, the value's Node is returned.

Example
const q = new Quercus();

q.setPath(["foo", "bar"], 5);
// => Quercus{"bar": 5}

q.setPath(["bar", "fazz"], 560);
// => Quercus{"fazz": 560}

q.setPath([], "foo");
// => null
